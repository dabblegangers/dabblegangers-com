# Dabblegangers

Creative outlet for a handful of awesome people who like to dabble.

Site for [dabblegangers.com](https://dabblegangers.com). Built from [`procrastivity/astro-bones`](https://github.com/procrastivity/astro-bones).

## Local dev

    nvm use
    npm install
    npm run dev

## Deploy

Pushes to `main` auto-deploy via Cloudflare Pages.
